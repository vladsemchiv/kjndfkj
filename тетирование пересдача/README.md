# Остров — пример готовой работы (Java + JUnit 5)

Это **пример** выполненного задания:

- Unit tests: **13** (10+ выполнено)
- Functional tests: **2** (прогон приложения через `Main` с подменой stdin/stdout)
- Maven + JUnit 5
- README с описанием тест-дизайна и запуском

## Как работает приложение

Программа читает команды из stdin (построчно):

- `SIZE N` — размер острова N×N (N >= 1)
- `START X Y` — стартовая позиция (в пределах острова)
- `MOVE U|D|L|R` — попытка сдвига на 1 клетку
- `REPORT` — печатает текущую позицию
- `EXIT` — завершение

Примеры вывода:

- `OK ...` — успешная операция
- `BLOCKED` — движение упёрлось в границу
- `ERROR ...` — некорректная команда/состояние

## Запуск тестов

Требования: Java 17+, Maven 3.8+

```bash
mvn test
```

## Тест-дизайн (как подбирались тест-данные)

Использованы техники:

- **Equivalence Partitioning**: валидные/невалидные классы команд и параметров
- **Boundary Value Analysis**: size=1/2, координаты 0 и size-1, выход за границы -1 и size
- **Negative testing**: пустая строка, неизвестная команда, START до SIZE, MOVE до START
- **State-based**: проверка изменения позиции до/после MOVE
